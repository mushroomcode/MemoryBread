# Mysql中的读写操作

Mysql中主要分为读-读、读-写、写-读、写-写几种操作。

### 1. 读写总结:  
- 快照读  
    使用MVCC进行快照读取。

- 锁定读
    - 行级锁：  
        - 共享锁（S锁，share）：
            select ... for update, select ... IN SHARE MODE。
        - 独占锁（X锁，exclusive)：
            update、insert等。
        S锁、X锁间互不兼容，S锁和S锁兼容，X锁间不兼容。
    - 表级锁：  
        - 意向锁
            - 读意向锁（IS）
            - 写意向锁（IX）
